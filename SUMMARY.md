   - All pages are linked via navigation

2. **To customize colors**:
   - Edit CSS variables in `assets/css/style.css` (lines 10-25)

3. **To add new pages**:
   - Copy structure from existing pages
   - Use the same navbar and footer
   - Apply `data-reveal` attribute for animations

   - Follow `SISTER_PAGES_GUIDE.md`
