# ARISE بخمسة

A faith-driven educational and healing institute website built with Bootstrap 5.

## Overview

ARISE بخمسة is a static website designed to provide a peaceful, trustworthy platform for mothers, families, and the Muslim community to access educational resources, counseling services, and community support.

## Features

- **Responsive Design**: Fully responsive layout using Bootstrap 5
- **Multilingual Support**: English and Arabic text support
- **Calm Aesthetics**: Soft neutral color palette with gentle animations
- **Service Pages**: Detailed information about library, counseling, homeschooling, and more
- **Sister Profiles**: Individual pages for each team member
- **Contact Form**: Simple contact functionality
- **Talent Platform**: Marketplace for showcasing community talents

## Technology Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- Google Fonts (Amiri, Cairo)

## Running Locally

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the site using the menu

No server setup required - it's a static site.

## Customization

- **Colors**: Adjust CSS custom properties in `style.css`
- **Fonts**: Modify Google Fonts links in HTML files
- **Content**: Edit text directly in HTML files
- **Images**: Replace placeholder image paths with actual images
